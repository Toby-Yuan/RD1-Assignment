# 題目
## 個人氣象網站
1. 縣市可選擇
2. 顯示當前天氣
3. 顯示未來天氣(2天, 1週)
4. 過去1小時, 24小時累積雨量
5. 切換顯示顯示圖片
6. 抓取資料放進資料庫

# 所需資料
## 一般天氣預報-36hr
1. 縣市名稱
2. 天氣敘述
3. 天氣敘述代號
4. 降雨機率

## 局屬氣象站-現在天氣
1. 縣市名稱
2. 鄉鎮名稱
3. 氣溫
4. 最高氣溫
5. 最低氣溫
6. 站名

## 雨量站-雨量觀測
1. 站名
2. 一小時降雨
3. 一天降雨
4. 本日降雨
